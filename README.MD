
# Rick and Morty API
A JSON API using mongoose and express. 


## API Reference

#### Get all items

```http
  localhost:3000/chracters
```




## Documentation

How to utilize characters RickandMorty API

1: run node seed/data.js

2: Run npm start

4 Access root localhost:3000/

Return characters localhost:300/characters
